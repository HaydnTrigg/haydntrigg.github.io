How to use?

Copy Reclaimed contents to halo_online\
Copy TagTool contents to halo_online\maps\
Copy Halo 3       to halo_online\maps\halo3_retail
Copy Halo 3: ODST to halo_online\maps\halo3_retail

The ordering of porting maps is important and different ordering will make multiplayer and co-op crash. Please follow this exact ordering. Please note that porting all of the content will take many hours.

Run the following commands below in TagTool:
opencachefile halo3_retail\maps\construct.map
PortTag Recursive Scripts MatchShaders *.scnr
exit

opencachefile halo3_retail\maps\010_jungle.map
PortTag Recursive Scripts MatchShaders *.scnr
exit

updatemapfiles halo3_retail\maps\info
updatemapfiles halo3odst_retail\maps\info
